Privacy Policy

Last updated: June 30, 2026

This Privacy Policy describes how JuPaAI Production ("we", "us") handles information in connection with our content publishing activities on YouTube, TikTok, and Facebook.

1. Information We Collect

We do not collect personal data from viewers of our content. Our automated publishing pipeline processes only content we create internally (AI-generated stories, audio, and video).

2. Google API and YouTube Usage

We use the YouTube Data API v3 and YouTube Analytics API solely to manage our own YouTube channel (@JuPaAI). Specifically, we use these APIs to:

  • Upload AI-generated videos to our own YouTube channel as private videos
  • Publish those videos to the public at scheduled times
  • Post comments on our own videos (affiliate links)
  • Manage playlists on our own channel
  • Retrieve basic channel statistics (subscriber count, view count) for internal monitoring

We access only our own channel data. We do not access, store, or process any data belonging to other YouTube users or channels. Our use of Google user data is limited to what is necessary to operate our own channel and complies with the Google API Services User Data Policy, including the Limited Use requirements.

We use OAuth 2.0 to authenticate with Google services. OAuth tokens are stored locally on our server and are never shared with third parties.

3. TikTok API Usage

We use the TikTok Content Posting API solely to publish AI-generated videos to our own TikTok account (@JuPaAiProduction). We do not access, store, or process any data belonging to other TikTok users.

4. Facebook API Usage

We use the Facebook Graph API solely to manage our own Facebook Page ("Jupaai True Crime"). Specifically, we use the API to:

  • Upload AI-generated videos to our own Facebook Page as Reels
  • Publish those Reels to the public at scheduled times
  • Retrieve basic Page statistics (follower count, view count) for internal monitoring

We access only our own Page data. We do not access, store, or process any data belonging to other Facebook users or Pages. Page access tokens are stored locally on our server and are never shared with third parties.

For questions about data deletion, please contact us at jupaai.business@gmail.com and we will process your request within 30 days.

5. AI-Generated Content

All published content is created using artificial intelligence. No real personal data or user-submitted content is used in our story generation process.

6. Third-Party Services

We use the following third-party services in our pipeline:

  • Google YouTube Data API v3 – Video upload and channel management
  • Google YouTube Analytics API – Channel statistics
  • Facebook Graph API – Video upload and Page management
  • Anthropic Claude API – AI text generation
  • ElevenLabs – Text-to-speech audio generation
  • TikTok Content Posting API – Video publishing

Each service operates under its own privacy policy.

7. Data Retention

We do not retain any viewer data. Generated content files are stored locally for operational purposes only. OAuth tokens and API access tokens are stored locally and refreshed automatically.

8. Contact

For privacy-related questions, contact us at: jupaai.business@gmail.com
