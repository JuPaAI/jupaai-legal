JuPaAI

JuPaAI is an automated content production system that creates and publishes AI-generated True Crime short videos to YouTube (@JuPaAI) and TikTok (@JuPaAiProduction).

The application uses the YouTube Data API and YouTube Analytics API exclusively to manage our own YouTube channel — uploading videos, scheduling publications, managing playlists, and monitoring channel performance. No third-party user data is collected or processed.
